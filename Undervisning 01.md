# Undervisning 01

Velkommen til utvikler kurs!

Fra og med i dag vil dette være et av dine viktigste verktøy i å lære web utvikling.


Her vil informasjon med material fra undervisninger og oppgaver fra undervisninger bli publisert, så det gjelder å holde et øye på oppdateringer på denne siden.


## Nettsider vi skal bruke gjennom kurset

- Google Classroom: som blir brukt for material og oppgaver igjennom kurset
- Github: som er en dokumentasjons platform som vi kan laste opp kode prosjekter for versjonskontroll og lagring
- Stackblitz: som er nettbasert programvare for å opprette prosjekter og skrive kode i
- Vercel: en hosting side for å publisere prosjekter
- Discord: som blir brukt til kommunikasjon for for kurset
- Figma: som er et design program

## Andre nettsider som kan bli relevant i kurset

- Unsplash: en nettside for gratis bilder
- Coolors: en nettside for fargepaletter
- Flaticon: en nettside for ikoner

## Temaer vi skal innom i løpet av de neste 12 ukene

- Semantisk HTML
- Plasserings metoder i CSS
- Animasjoner i CSS
- Design planlegging
- Nettside analyser
- Responsive nettsider
- JavaScript for CTA funksjonaliteter
- Data lagring og data henting med JavaScript
- Det kan skje endringer i hvilke nettsider, programvarer og temaer vi skal innom.
